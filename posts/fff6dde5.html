ps学习

1.认识主界面

选项栏是调节工具相关参数的面板，位置和显示都可以调节

“Ctrl + 鼠标滚轮” “Shift+ 鼠标滚轮” 也可以实现画板移动

“Alt+ 鼠标滚轮” 可以放大可缩小画面

画面周围灰色的区域为文档的工作面

选择油漆桶工具并设定前景色

按住“Shift” 点击灰色工作面区域

2.新建文档

点击菜单栏-文件-新建，可以打开新建选项（快捷键 Ctrl + N）

剪贴板

未采集像素则不可用，选框为灰色，不能点用剪贴板截图，打开ps,系统会自动计算该截图尺寸，直接建立应该相同大小文档，或直接使用该剪贴图
或者直接去复制—张图片，也能激活剪贴板，建立相同大小文档，粘贴图片即可使用
如果是文件夹里复制—张图片来平时，是不能激活剪贴板功能的，这里的复制是文件整体的复制，剪贴板是复制图片的数据

Photoshop数值输入框前的文字都可以拖动，都可以通过鼠标拖拽来改变数值大小

像素和矢量图

“像素” 是计算数码影像的一种单位，1个像素就是最小的图形的单位，在屏幕上显示的通常就是单个的染色点，放大会失真，边缘锯齿化

矢量图没有像素的概念，它是面向对象的图像或绘图图像，在数学上定义为一系列由线连接的点

网络图片，网页或者软件界面 视频图片等显示视频上使用的图片就要以像素为单位。

分辨率

分辨率的单位简称Ppi，也就是Pixel Per Inch。72Ppi即表示1英寸由72个像素点组成

分辨率越大，像素越多，图像的细节也就越多自然就越清楚

颜色模式

新建文档中的【颜色模式】包括【位图】【灰度】【RGB颜色】【CMYK颜色】【Lab颜色】5种

RGB:红绿蓝三种颜色叠合，按不同比例混合能呈现16777216种颜色（8位深度），是广泛用于显示屏的一种基本色彩模式，大多为网页，网图，视频，3D贴画等电子图像

CMYK:由青色，品红，黄色，黑色油墨进行混合，来表现初各种印刷颜色，由于是印刷色不是发光色，在显示器上会看起来黯淡一些，多用于海报，宣传册，单页等印刷图像

位数，常规选8位，16位/32位是专业研究用的

背景内容—背景色

点击左下角小屏幕标识可恢复为默认色，也可按D
点击右上角箭头小标识可切换前景背景色，也可按X

按住Alt键不松，“取消”按钮变成了“复位”

修改图像大小

快捷键：Ctrl + Alt + i

像素的多少决定了文件的大小

文档大小是修改文档的尺寸，文档尺寸改变需要结合分辨率

勾选缩放样式修改文档大小时，它的图层样式也会跟着放大缩小，是成比例的。

约束比例意思是限制长宽比，选择后高度和宽度后面会出现锁定框（修改宽度的数值高度也会随比例改变）

取消重定图像像素图像大小会被锁定，只能修改文档大小

为了文件用于印制所以调整文档尺寸也很重要，分辨率的单位是像素/英寸，在像素固定的情况下修改分辨率高度宽度也会变化

画面总像素值=宽度像素值×高度像素值

画面总像素值=高（英寸）×宽（英寸）×分辨率（像素/英寸）²

对于网络图片/显示屏图片的分辨率设置为72或96

挂网也有一个精度指标，叫做线分辨率，简称LPI。LPI是印刷图象所用的网屏的每英寸的网线数，也就是挂网网线数。

3.文档的打开与保存

打开

1.文件>打开

2.双击舞台

3.快捷键Ctrl + o

勾选图像序列，可以选择命名上有次序的多个图象

保存

Ctrl + S 存储（原地存储）

Shift + Ctrl + S 存储为（可以选择存储位置、存储名称和存储格式）

PSD是Photoshop的标准文件格式，包含颜色、图层、通道、路径、动画等信息，是创作图像作品的原始文件。可以说有Photoshop的地方，必有PSD文件

JPG是最流行的图片文件格式，体积小巧、可变压缩比、支持交错，广泛用于互联网传输，是我们最常见的图片格式

GIF图片支持透明色、支持动画，网页上看到的动态图、聊天表情大都是GIF图片

Shift + Ctrl +Alt + S 存储为Web所用格式

关闭

Ctrl + w 关闭

Ctrl + Alt + w 关闭全部

4.前期准备

暂存盘

找到首选项，选择性能，快捷键 Ctrl + K

Photoshop在工作的时候，会产生临时文件，因为软件在运算的过程中，会产生大量的数据，要把数据暂时存在硬盘空间上。一般情况下，【暂存盘】默认设置为第一个驱动器。对于Windows系统来说，也就是C盘，取消C盘，改为其它空闲盘

为了方便操作失误后，找回更多的历史记录，我们可以将历史记录状态选项调大一些

自动保存设置

首选项>文件处理

自动存储恢复信息时间间隔 改为5分钟

PS的自动保存可以在后台运行，并不拖慢前台的处理速度。

快捷键设置

有的命令经常用，但却没有快捷键

有的快捷键按键太多，一个手几乎按不过来

可以自定义快捷键【编辑】>【键盘快捷键】

注意：如果不是特殊工作需要，请勿修改快捷键

5.图层基础知识

认识图层

1.菜单栏上的图层菜单

2.面板栏的图层面板（窗口>图层 调出 快捷键F7）

图层锁定

填充是对图像像素填充度的调节

图层列表，图层是按上下叠加的次序来排列的

底栏按钮

列表下方还有一栏按钮

从左往右依次是链接图层按钮，图层样式按钮，添加蒙版按钮，添加调整图层按钮，创建新组按钮，创建新图层按钮，删除按钮。

图层理解

图层简单的来说就是图像的层次

一个空白图层就像是一个透明胶片，可以很方便的单独调整和修改图层。

图层类型

  • 普通图层
  • 背景图层
  • 智能对象图层
  • 调整图层
  • 填充图层
  • 视频图层
  • 矢量图层
  • 3D图层
  • 文字图层

创建图层

底栏有新建图层按钮，点击可创建新图层

【图层】>【新建】>【图层】快捷键 Ctrl + Shift + N

可以选择图层的识别颜色，并不是图层上面像素颜色

双击图层名字能修改图层名称，只有双击文字才能修改名字，否则会弹出图层样式窗口

图层操作

选择、隐藏、显示、移动、复制和删除

可以结合Ctrl和Shift键，一次性选择多个图层

点击小眼睛，来打开和关闭图层像素的显示

存jpg或者其它最终格式的图像的时候，隐藏图层不显示

可以通过快捷键隐藏目标图层以外的图层，快捷键Alt

图层上下次序改变后，画面上的叠加次序也就发生变化了，快捷键Ct + [ 可以把选中的图层向下移动一层，快捷键Ct + ] 可以把选中的图层向上移动一层

Ctrl+Shift+左中括号，把选中的图层直接移动到最下方

Ctrl+Shift+右中括号，把选中的图层直接移动到最上方

【图层】>【复制图层】进行复制

按住alt键，来拖拽某个图层也可以复制图层，鼠标拖拽的位置就是复制的位置

【图层】>【新建】>【通过拷贝的图层】意思是通过对下方图层的拷贝新建一个下方图层一样的图层

快捷键Ctrl+j

6.平移和缩放

在工作信息栏左下角可以改变画面大小

放大、缩小、旋转、平移、只是对视图进行调整，并不是对图像的像素进行修改

工具栏下面小手一样的图表是抓手工具，选择抓手工具，鼠标在画面就会变成一个小手，这个小手就是用于移动画面视图的，在画面上点击拖动，就可以自由的移动画面

菜单栏的下方是选项栏，是对工具进行调节的

按住空格键不松可以直接转换为抓手工具，松开空格会返回到当前工具

在变成抓手工具的同时按住Alt就变成了缩小工具

按抓手工具不松，会弹出折叠在里面的一个工具，旋转视图工具，快捷键是R，按住鼠标不松，进行拖动，旋转工具可以对视图进行旋转

在选项栏可以自定旋转的角度，也可以拖拽属性名称来修改数值，点击复位视图，图像就回到了原始状态

旋转视图工具主要用途是与数位板相结合

缩放工具可以对画面进行放大或缩小，在选项栏有放大缩小的选择，也可以点击鼠标右键对画面进行放大或缩小

细微缩放：点击图像左右移动鼠标对视图进行缩放

放大快捷键：Ctrl + ‘’加号“ 缩小快捷键：Ctrl + ‘’减号“

Ctrl + 0 是显示图像全部

Ctrl + Alt + 0 是显示图像实际大小

空格键+Ctrl键是放大工具，空格键+Alt键是缩小工具

在窗口菜单里找到导航器，点击它，导航器下方的滑杆也可以控制画面的大小，也可以输入数值改变画面大小

7.移动工具

移动工具 快捷键 V

移动工具可以多文件间拖拽图层对象，拽至标签激活目标文档，并拽入，原文档并没有移走

拖拽位置根据鼠标位置而定

如何移动同时保持原位置不懂呢？

文档大小相同，开始拖动后，按下Shift，可保持原位

文档大小不同，开始拖动后，按下Shift，会自动居中，如果目标文档包含选区，会到选区中央

选项栏上的工具和设置

自动选择可以靠点击图层对象来选择图层

自动选择状态下，在透明区域可框选

移动工具状态下，可以按Ctrl快速切换自动选择模式

其它工具状态下，按下Ctrl快速切换移动工具

组合Alt键，可以复制图层

组合Shift键，约束角度

方向键可以微调，组合Shift，步长加大

方向键结合Alt键是复制图层

创建新组按钮可以建组，可以选择组进行检索

图层菜单里图层编组可以快捷的建组，快捷键：Ctrl + G

勾选自动选择，然后选择组，然后选择组里的任何一个图层，都会选择组，改为图层后，就这能选择图层，而不能选择组

选中显示变换控件，就能对图层对象进行变换

后面的工具是对齐与分布方式和自动对齐图层

【图层】>【对齐】和此时的选项都是一样的，都是针对于图层来操作的。

顶对齐是以画面最上方图层的顶线为基准的对齐

垂直居中对齐是以图层垂直方向的中心点进行对齐

底对齐是以画面最下方图层的底线为基准的对齐

左对齐是以画面最左边图层的左边线为基准对齐

水平对齐是以图层的水平中心点位基准对齐

右对齐是以画面最右边图层的右边线为基准对齐

默认情况下，目标侧的图层固定，其他图层移动找齐

图层链接后就会以选中的图层为基准对齐

按左分布是以图层左边线为基准分布

第一点：2、3、4要在1和5的范围内

第二点：可以把图层打上链接

8.选区和选框工具

在进行图像的编辑时，要对图层中某部分的像素进行处理，要把这部分单独选择出来，这个部分叫做选区，在PS中，选区表现为一个封闭的游动虚线区域，因为看上去像是一圈爬动的蚂蚁，俗称蚂蚁线。

选区内像素可被编辑，可被移动

矩形选框工具，快捷键M，光标显示方式为十字

点击选区，按住Shift键，可进行正方形选区

按住Alt键，可以从中心点建立选区

同时按住Alt和Shift键，从中心点建立正方形选区

矩形选区包含了所有类型选区的属性，鼠标进入选区变成白箭头可以移动选区

Ctrl+D取消选区，Ctrl+Shift+D重新选择

Ctrl+H可以隐藏蚂蚁线显示，选区仍存在

历史工具也可以找回选区，”重新选择“记录上一步选区

选区是暂时性的，只有保存选区后才能被存储到PSD文件，在通道面板可以找到被保存的选区

9.选框工具全解

选区有四种组合方式

  1. 新选区
  2. 添加到选区（在新选区模式下按Shift键切换到添加到选区，先按住Shift键后再选区）
  3. 从选区减去（在新选区模式下按Alt键切换到从选区减去）
  4. 与选区交叉（在新选区模式下按Shift+Alt键切换到与选区交叉）

羽化就是选区边缘部分实现过渡式虚化，从而起到渐变的作用，从而达到选区内外自然衔接的效果

羽化值的大小决定虚化程度

也可以先做普通选区，然后再进行羽化【选择】【修改】【羽化】快捷键Shift+F6

矩形选区不存在锯齿

固定比例，锁定宽高比，固定大小，锁定大小

调整边缘，必须要有选区

选中椭圆选框工具，消除锯齿被激活，不选择消除锯齿图像边缘会有锯齿，选择则会变得柔和

单行和竖行选区工具属于特殊的矩形选框

单行和竖行选区是可以羽化的

警告：任何像素都不大于50%选择。选区边将不可见。

此情况下，选区不可见，仍然存在

10.套索与魔法工具

套索工具也叫自由套索工具，快捷键是L

用法：点击鼠标左键拖动鼠标建立选区

套索工具灵活快速但不够准确

套索工具的选项栏是选区类工具通用的参数

按Shift+L键或者按着Alt键来切换

多边形套索的用法，点击画面的关键的点，来直线连接（在选区未闭合情况下，鼠标不能点选其他命令）

我们可以通过Backspace键或Delete，来删除刚才的点

Esc取消选择

首尾接近时光标出现小圆圈，可以点击闭合（交接快捷方式双击、按住Ctrl左击、或者回车）

多边形套索工具可以画到文档的外面，而自由套索则不行

套索工具和多边形套索工具可以按着Alt键相互切换

多边形套索适合于选取直线边缘（可以结合放大、缩小、平移，来进行多边形选区制作）

磁性套索的控点可以智能识别像素边缘，磁性套索可以通过Alt键，与多边形套索快速切换。宽度的大小控制识别的范围，高对比度适合清晰边缘，低对比度适合模糊边缘，频率来调节磁性控点的多少

魔棒工具可以快速的将颜色相近的区域变成选区，容差越大识别范围就越大，反之就越小；勾选连续只能选取互为相邻的相似像素；勾选对所有图层取样，可以选择没有选中图层的颜色

快速选择工具，三种模式是做选取类通用的模式，但是它没有交叉模式；画笔拾取器包含大小，硬度，间距，大小代表识别的范围（快捷键左、右中括号），硬度是边缘的识别能力（快捷键Shift+左、右中括号），间距是识别的连贯程度；勾选自动增强，识别边缘的能力就更强

调整边缘命令可以优化选区

11.选取的编辑调整

选取的自由变换

鼠标右击找到变换选区，或在选择菜单中找到变换选区，此时出现的控杆控点跟前面讲到的变换控件是一样的，拖动控点对选区进行变形，控杆也可以进行这样操作。鼠标右键，打开右键菜单可以对变形进行很多操作

组合快捷键或调整上方选项栏参数对选区进行准确调节，变换完后，点击对号，确认变换结束，点击别的工具也可以完成选区变换的确认；如果选区变换的不是很满意，点击取消符号或者Esc键取消选区

反向选择

选取背景后，选择反向，留住主体；通过右键菜单，右击，选择反向；选择菜单中也可以找到反向命令，快捷键Ctrl+Shift+I

羽化

羽化可以制作出边缘比较柔和的选区

羽化使选区内透明度值向外递减

透明度

两个外形一样选区，透明度却可以不同

通过填充颜色，来看出两个选区透明度的差别

填充前景色，快捷键Alt+Delete

半透明选区制作：

打开通道面板，新建通道

在通道里制作一个选区，填充灰色

Ctrl+D取消选区

在通道面板点击按钮：将通道作为选区载入

边界

边界命令可以让边界变成选区

扩展与收缩

平滑命令可以平滑尖角

扩展与收缩来对选区进行周边缩放

选区转路径

选区可以转换为路径

路径是重要的辅助工具，是矢量元素

12.历史工具

当选区出现错误时，使用历史工具返回

找到窗口下的历史记录点击后出现历史记录面板

最上方为原始图片，也可以叫做快照栏，下面出现的操作步骤栏，为历史记录栏，面板下方还有三个按钮

红+绿=黄 红+蓝=品红 绿+蓝=青

CMYK模式详解

印刷色彩模式

CMYK通道里，黑色代表100%浓度的油墨

CMYK靠百分比来表达当前通道的色值

RGB是加色模式，CMYK是减色模式

20.亮度与色阶看懂直方图

控制明暗的视觉因素，三要素之一，明度

明度较低时，RGB色值偏低，CMYK色值较高

明暗对比：明靠暗衬托，暗配明更暗。

Ctrl+U 色相/饱和度也能调节明度，但效果太单一

灰度模式的图像，只有亮度的对比，更能够直观观察明暗

灰度模式，好比CMYK通道中的单个K通道

在灰度模式下，使用亮度/对比度命令

调节滑块改变数值，相应的发生明暗变化

亮度/对比度是最简单的调色命令，功能有限

Ctrl+L 色阶

色阶，合并颜色的发光级别

直方图：用二维坐标表示画面像素发光强度的分布

水平方向从左到右依次为：黑场，灰场，白场

垂直方向上的高度，代表着像素的数量

调整输入色阶

黑场数值增大，画面变暗

亮度级别为70以下的像素，都合并为最低的0

亮度级别70以上，至灰场以下的像素，都会阶梯式变暗

70以下的像素亮度都归0，所以暗部图像的细节不再有

调整黑场，就是合并暗部的颜色，对亮部暂无影响

调节白场，和黑场的道理一样，反向去理解

调节灰场，是调节的黑白场的比例

调整输出色阶

输出色阶，对最高级别和最低级别，做一个输出限定

预设里提供了一些设定好的参数，也可以保存预设。

合并红通道的黑场，红色减少，颜色开始偏蓝绿

合并红通道白场，红色增强，亮部发红

色阶里合并单个通道，也是调色手段，按情况使用即可

自动是PS根据图像，自己做出的调整，一般不会过度

调整类的命令，会损失原图像素的色彩信息，不能灵活修改

调整图层

在调整图层可以找到和对应的调整命令相同的参数

调整图层，对下方所有的图层起着调整作用

调整图层可以即调即得，随时修改参数，改变图像效果

调整图层不对下方图层的像素做任何修改破坏

调整图层自带蒙版，可以灵活控制调节或者不调节的区域

调整图层对其下方所有的层起作用，对上方的没有影响

调整图层结合剪贴蒙版，可以对单个图层起调整作用
